# SIV Monitor

## Présentation

SIV Monitor est un mini outil de supervision d’un système d’information voyageurs embarqué.

Le projet simule un système technique présent dans un environnement ferroviaire et permet de surveiller plusieurs paramètres :

- température système
- connexion
- alimentation
- écran voyageurs
- GPS
- annonces sonores

## Objectif

L’objectif est de montrer une approche simple de supervision, d’automatisation et de détection d’anomalies sur un système embarqué.

## Fonctionnalités

- Génération automatique de données techniques
- Détection d’anomalies
- Enregistrement des logs dans un fichier CSV
- Génération d’un rapport de fiabilité
- Dashboard Streamlit
- Graphique de température
- Tableau des alertes

## Technologies utilisées

- Python
- Streamlit
- Pandas
- Matplotlib
- CSV
